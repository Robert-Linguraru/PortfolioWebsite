
##  Portfolio Website

Welcome to the repository for my personal portfolio website\! This project showcases my skills, experience, and projects as a developer.

###  Key Features

  * **Responsive Design:** Optimized for seamless viewing across all devices (desktop, tablet, and mobile).
  * **Performance Focused:** Built with modern practices for fast loading times and smooth transitions.
  * **Interactive UI:** Enhanced user experience with delightful animations powered by **Framer Motion**.
  * **Type Safety:** Developed using **TypeScript** for robust, scalable, and maintainable code.
  * **Modern Styling:** Utilizes **Tailwind CSS** for rapid and utility-first styling, (favourite part being no specificity issues :) ).

-----

###  Tech Stack

This project is built using the following technologies:

| Category | Technology | Description |
| :--- | :--- | :--- |
| **Frontend Framework** | **React** | The core library for building the user interface. |
| **Language** | **TypeScript** | Adds static typing to JavaScript for enhanced code quality. |
| **Styling** | **Tailwind CSS** | A utility-first CSS framework for custom design. |
| **Animation** | **Framer Motion** | A production-ready library for declarative animations in React. |
| **Deployment** | [**Vercel**] | Where the website is hosted and deployed. |

Feel free to fork this project and use it as a base for your own portfolio.
